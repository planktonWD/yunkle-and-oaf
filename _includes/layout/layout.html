Layout

Breaking this up into sections are quite a few handy class available

Containers

.layout wraps .layout__item and widths if needed are added to the .layout__item as a sepearate class following the single responsibility priciple.

.layout pushs the container to the left -$spacing-unit and a padding is added to the children .layout__item(s)

!important note! - please comment out any whitespace between .layout__item(s) this is the 1px inline-block issue - this resolves this - I know not ideal but better that others. See http://csswizardry.com/csswizardry-grids/ (note the difference is that "layout" is used in this styleguide rather than "grid")

Responsive Layout

The following are a list of breakpoints, used them with the width class together to inforce a width at a breakpoint very handy.

"max-mobile" "screen and (max-width: 540px)",
"min-mobile" "screen and (min-width: 500px)",
"two-hands" "screen and (min-width: 700px)",
"small-lap" "screen and (min-width: 800px)",
"lap" "screen and (min-width: 860px) and (max-width: 1047px)",
"lap-and-up" "screen and (min-width: 860px)",
"portable" "screen and (min-width: 900px)",
"desk" "screen and (min-width: 1072px)",
"below-desk" "screen and (max-width: 1071px)",
"retina" "(-webkit-min-device-pixel-ratio: 2), (min-resolution: 192dpi)"

For example: div class="layout__item two-hands-one-half" would make an element inline block full width on mobile but at min-width 700px breakpoint if would be width 50%

Utility Classes

These classes "trump" everything so very useful for enforcing a look without writing custom CSS for a non-modular piece of markup.

(m|p)(t|r|b|l|)(-|+|0) {}` = margin/padding top/right/bottom/left

Example div class="u-pr" adds $spacing-unit padding right to element. div class="u-mb0" makes elements margin-bottom: 0.

Also there are vertical alignment and border classes

.bb adds a border bottom etc
.u-vb vertically aligns element to bottom
.u-tt makes text top

These are mainly used for aligning buttons sometimes

Helper Classes

Also available are .alignleft, .alignright, .aligncenter which float elements and add margins to them. You can tweak margins with the utility classes above
