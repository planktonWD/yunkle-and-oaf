Select Elements

The jQuery plugin works it's magic on select elements with a class of either .single-select or .multi-select. Inspect one here to see and also note the use of .formfield--svg on the container so position the down arrow and the svg tag is required also.

Time and Date Fields

The jQuery plugin works it's magic on select elements with a class of either .timepicker or .datepicker as above make sure you add .formfield--svg to the container and pop the calendar svg in there too
